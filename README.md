# 🐍 Snake Game (Mobile + Desktop)

A fun and interactive **Snake Game** built using **HTML, CSS, and JavaScript**.
This game supports both **keyboard controls (desktop)** and **touch controls (mobile)**.

---

## 🚀 Features

* 🎮 Smooth snake movement with increasing score
* 📱 Mobile-friendly touch controls (Up, Down, Left, Right buttons)
* ⌨️ Keyboard controls using arrow keys
* 🔊 Sound effects (food, movement, game over, background music)
* 🏆 High score saved using local storage
* 🎨 Attractive UI with background and animations

---

## 📁 Project Structure

```
├── index.html     # Main HTML file
├── style.css      # Styling and layout
├── script.js      # Game logic
├── food.mp3       # Food sound
├── gameover.mp3   # Game over sound
├── movebtn.mp3    # Movement sound
├── bg2.mp3        # Background music
```

---

## 🕹️ How to Play

### 💻 Desktop

* Use **Arrow Keys**:

  * ⬆️ Up
  * ⬇️ Down
  * ⬅️ Left
  * ➡️ Right
